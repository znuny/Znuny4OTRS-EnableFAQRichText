Znuny4OTRS - EnableFAQRichText
==============================
This package enables richtext for FAQ even if richtext is disabled in OTRS.


**Installation**

Download the package and install it via admin interface -> package manager.


**Prerequisites**

- OTRS 3.0

- OTRS 3.1

- OTRS 3.2

**Configuration**

Not needed.
